```
src/app/shop/         → Ecommerce (Phase 2)
src/app/privacy/      → Privacy Policy
src/app/terms/        → Terms of Service
src/components/       → Shared components (Header, Footer, etc.)
src/lib/              → Utilities (email, db, etc.)

public/icons/         → SVG icons

```
